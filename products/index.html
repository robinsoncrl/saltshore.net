CalGen

iCalendar generation that actually works.

The iCalendar standard (RFC 5545) is 147 pages long and broken by every major calendar tool in a different way. CalGen generates compliant ICS files by construction — no timezone drift, no silent field drops, no property ordering failures.

  • Canonical VTIMEZONE blocks and IANA timezone identifiers
  • Correct RRULE, EXDATE, and RECURRENCE-ID handling
  • Stable UIDs — same event never becomes a duplicate on re-export
  • Deterministic output — identical inputs produce identical bytes
  • Local audit log of every generation event

FinPro

From structured time to a defensible invoice.

Freelancers lose 10–20% of billable income annually to tracking gaps — not low rates, not slow clients, but invisible gaps between hours worked and hours billed. FinPro closes that gap by ingesting CalGen time entries directly and calculating every invoice from explicit rules.

  • CalGen time entries flow in via .finpro.json — no manual transcription
  • Versioned rate definitions with effective_from and effective_to dates
  • Overtime, weekend, and holiday multipliers applied automatically
  • Immutable line items — every amount locked at invoice issuance
  • Transparent gross-to-net deduction breakdown
  • Household Mode for chore tracking and allowance calculation

LedgerPro

Clean financial records — without the cloud.

When Mint shut down in early 2024, 3.6M users lost years of financial history. LedgerPro takes the opposite approach: file-based import, local SQLite storage, and a deterministic engine that never silently resolves ambiguity.

  • Configurable bank profiles — normalize any CSV or OFX format
  • Composite duplicate detection (amount + date + description fingerprint)
  • Transfers as first-class entities — excluded from income and expense totals
  • Canonical merchant alias groups with visible rule derivation
  • Statement coverage index — gaps surfaced before tax time
  • SHA-256 document integrity hashing on every imported file
  • FinPro invoice reconciliation — matches deposits to issued invoices
