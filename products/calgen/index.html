CalGen

The iCalendar standard (RFC 5545) is 147 pages long and broken by every major calendar tool in a different way. CalGen generates compliant ICS files by construction — no timezone drift, no silent field drops, no property ordering failures.

What it does

  • Canonical VTIMEZONE blocks and IANA timezone identifiers
  • Correct RRULE, EXDATE, and RECURRENCE-ID handling
  • Stable UIDs — same event never becomes a duplicate on re-export
  • Deterministic output — identical inputs produce identical bytes
  • Local audit log of every generation event

Where it fits

CalGen is the first stage of the SaltShore pipeline. Time entries structured in CalGen flow directly into FinPro — no manual transcription, no copy-paste errors.
